This is a IO Guide for working in low level Assembly and C language with GCC, LD and NASM. Most of the informations are based on various sources but not limited to OS DEV Wiki

GCC Compiler Suite

We will be using GCC Compiler Suite to compile our C files to produce the Object files. Normally when we use the default GCC that comes with the system to compile our C Program. Its perfect for programming softrwares that run within the same OS but is limited to only that perticular host in which the program is compiled.
Therefore, we require a special compiler, knows as

GCC Cross Compiler
to compile the program that runs on another host.
